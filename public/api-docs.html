API של המערכת

כדי שהבקשות יעבדו צריך להיות מחובר/ת באותו דפדפן — אם עדיין לא התחברת, היכנס/י כאן קודם, ואז חזרי לעמוד הזה (אפשר להשאיר את שתי הכרטיסיות פתוחות). הבדיקה כאן שולחת בקשות אמיתיות שמשנות נתונים במערכת.

מפתחות API

מפתח API מאפשר לכלים חיצוניים (כמו n8n) לגשת למידע ולעדכן אותו בלי להתחבר עם שם משתמש וסיסמה בכל פעם. יש לשלוח אותו בכותרת הבקשה: Authorization: Bearer <המפתח> (או X-API-Key: <המפתח>).
